使用说明

基本用法

将您的目标请求地址作为参数传递给代理端点即可：

[METHOD] /api/proxy?url=URL_ENCODED_TARGET
核心规范
  • 支持所有标准 HTTP 方法
  • 自动转发 Content-Type 与常用 Header
  • 自动处理 3xx 重定向
使用限制
  • 单次请求体最大限制为 1MB
  • 默认频率限制：100次 / 15分钟
  • 仅允许代理 HTTP/HTTPS 协议
  • 部分受限域名可能已被列入黑名单
JavaScript Fetch
// 发送 GET 请求
const getRes = await fetch('/api/proxy?url=' +
    encodeURIComponent('https://api.example.com/data'));
const data = await getRes.json();

// 发送 POST 请求
const postRes = await fetch('/api/proxy?url=' +
    encodeURIComponent('https://api.example.com/save'), {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ name: 'test' })
});
Python Requests
import requests

# 代理基础地址
proxy_url = "http://your-domain.com/api/proxy"
target = "https://api.example.com/data"

# 发送请求
response = requests.get(
    proxy_url,
    params={"url": target}
)
print(response.json())
